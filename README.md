## Project Overview

This Excel dashboard helps managers track sales performance, compare targets vs achievements, and identify trends across regions and product categories. It replaces manual spreadsheet analysis with a clear, visual decision-making tool.

## Problem Statement

Raw Excel sheets made it slow and error-prone to analyze sales performance. There was no consistent way to measure targets, trends, or regional performance, leading to delayed and unclear decisions.

## Data Description

* **Source:** Internal sales logs (`Sales Dashboard.xlsx`)
* **Data Size:** Monthly sales records with hundreds of rows
* **Key Fields:** Sales Amount, Units Sold, Sales Target, Product Category, Region, Month
* **Cleaning:** Removed duplicates, standardized names, fixed formats, and used formulas like `IFERROR`, `TRIM`, and `ISBLANK`

## Target Users

* Sales Executives
* Regional Managers
* Business Analysts
* Leadership Teams

## Dashboard Features

* KPI cards for total sales, targets, achievement rate, and units sold
* Monthly sales trend line charts
* Region-wise bar/column charts
* Product category performance analysis
* Conditional formatting for quick performance comparison
* Interactive slicers and dropdown filters

## Tools & Techniques

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Conditional Formatting
* Data Validation (dropdowns)
* Named Ranges
* Structured multi-sheet layout

## Scope & Limitations

**Scope**

* Monthly sales analysis by region and category
* Target vs achievement tracking
* Management-ready summaries

**Limitations**

* Manual data refresh
* No real-time collaboration
* No customer demographic analysis

## Key Outcomes

* Identified top and underperforming regions
* Tracked sales vs targets clearly
* Highlighted best-selling categories and seasonal trends
* Reduced manual reporting effort
